# AmazonUI

Amazon-style e-commerce front-end built with **HTML, CSS, and JavaScript**.

## Live Demo

https://oladegs.github.io/AmazonUI/

## Features

- Product listing with dynamic rendering from JSON
- Cart management (add/update/remove)
- Checkout with delivery options, tax + shipping calculations
- Order history and tracking pages
- Basic testing for core shopping workflows
